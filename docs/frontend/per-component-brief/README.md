# Frontend Component Briefs

Component briefs document reusable React component contracts. They cover shared UI primitives, layout shells, data display components, and domain workflow components that appear across page briefs.

Component briefs may describe data needs, but backend gap IDs live only in page briefs.

## Shared HTML Reference Boards

- `docs/frontend/html-reference/Shared - 05 - Layout Shells.html`: student, staff, Super Admin, footer/header, and auth shell references.
- `docs/frontend/html-reference/Shared - 06 - UI Primitives.html`: buttons, form controls, status badges, checkbox rows, and rating input.
- `docs/frontend/html-reference/Shared - 07 - Reservation Workflow Components.html`: stepper, reservation summary, document/payment/upload panels, and status panel.
- `docs/frontend/html-reference/Shared - 08 - Data Display Components.html`: facility card, category shortcut, filter bar, mobile card/table conversion, KPI, activity, profile, and governance rows.
